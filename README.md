# Library Management System

A software system for managing library books and members.

## Construction Decisions
-**Language:** python 3.13+
-**Testing:** pytest
-**Version Control:** Git

## Project Structure
-**src/** - Source code files
-**src/models/** - Contains the book model
-**src/services/** - Contains the library service logic
-**src/utils/** - Contains helper classes like member
-**tests/** - Unit testing files
-**docs/** - Documentation

## Language Decision Justification
The detailed explanation for selecting Python in this project is included in:

docs/Library Management System.docx

## Getting Started
1. Clone the respository
2. Install dependencies: `pip install -r requirements.txt`
3. Run tests: `pytest`
